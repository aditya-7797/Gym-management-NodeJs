A simple and lightweight **Gym Management Application** built using **Node.js, Express, and Vanilla JavaScript**.

It allows basic member management, check-ins, and demonstrates real-world concepts like API handling, cookies, and browser debugging.

---

## 🚀 Features

- 🧾 Member Registration  
- ✅ Member Check-in System  
- 🌐 REST API Integration (Fetch/AJAX)  
- 🍪 Cookie-based Theme Support  
- ⏱️ Live Timing Functionality  
- 🛠️ DevTools Debugging Practice  

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express  
- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Concepts Used:**  
  - REST APIs  
  - Cookies  
  - Async JavaScript (Fetch/AJAX)  
  - Browser Developer Tools
